# 出库管理版本字段添加说明

## 功能概述
在出库管理的创建/编辑出库单页面添加了"版本管理"字段,该字段关联到版本管理表,允许用户在创建出库单时选择对应的产品版本。

## 修改内容

### 1. OutboundFormPage.tsx (出库单表单页面)
- **导入版本相关函数和类型**:
  - 添加 `getVersions` 函数导入
  - 添加 `Version` 类型导入

- **状态管理**:
  - 添加 `versions` 状态存储版本列表
  - 添加 `versionId` 状态存储选中的版本ID

- **数据获取**:
  - 在 `fetchData` 函数中并行获取版本数据
  - 编辑模式下从订单数据中加载版本ID

- **表单字段**:
  - 在"客户"字段后添加"版本管理"下拉选择框
  - 显示格式: "版本号 - 版本名称"
  - 支持选择"无"选项

- **数据保存**:
  - 创建出库单时保存 `version_id`
  - 更新出库单时更新 `version_id`

### 2. OutboundPage.tsx (出库单列表页面)
- **表格列**:
  - 在"出库日期"和"状态"之间添加"版本号"列
  - 显示关联版本的版本号
  - 未关联版本时显示"-"

- **表格调整**:
  - 更新 `colSpan` 从6改为7以适应新列

## 数据库字段
- 表: `outbound_orders`
- 字段: `version_id` (UUID, 可选)
- 关联: `versions` 表的 `id` 字段

## 用户体验
1. **创建出库单**:
   - 用户可以在基本信息区域选择版本
   - 版本字段为可选,可以选择"无"

2. **编辑出库单**:
   - 已保存的版本信息会自动加载
   - 可以修改版本选择

3. **查看出库单列表**:
   - 列表中直接显示版本号
   - 便于快速识别出库单对应的产品版本

## 技术细节
- 使用 React Hooks 管理状态
- 使用 shadcn/ui Select 组件实现下拉选择
- 数据通过 Supabase API 与数据库同步
- 支持版本信息的关联查询显示

## 测试建议
1. 创建新出库单时选择版本
2. 创建新出库单时不选择版本(选择"无")
3. 编辑已有出库单修改版本
4. 在出库单列表中查看版本号显示
5. 验证版本信息正确保存到数据库
